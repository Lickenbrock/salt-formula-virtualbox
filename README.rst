
==========
VirtualBox
==========

VirtualBox is a general-purpose full virtualizer for x86 hardware, targeted at server, desktop and embedded use.

Sample pillars
==============

VirtualBox version 4.3

.. code-block:: yaml

    virtualbox:
      host:
        enabled: true
        version: 4.3
        extensions: false

VirtualBox version 5.0

.. code-block:: yaml

    virtualbox:
      host:
        enabled: true
        version: 5.0
